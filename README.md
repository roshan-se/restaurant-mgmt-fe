# RestoMate - Hospitality Management System

*A unified solution for modern restaurants and hotels*

## 🚀 Overview
RestoMate is an **all-in-one management system** combining:
- 💳 **Point-of-Sale (POS)**
- 🏨 **Property Management (PMS)**
- 📊 **Inventory & Analytics**
- 📱 **Customer Relationship Tools**


## ✨ Key Features
| Module          | Highlights                          |
|-----------------|-------------------------------------|
| **Smart POS**   | Tableside ordering, split bills, kitchen display |
| **Hotel PMS**   | Room assignments, housekeeping, OTAs integration |


# 🚀 Quick Start Guide

Get RestoMate up and running in 5 minutes:

## Installation
```bash
# Clone repository
git clone https://github.com/roshan-se/restaurant-mgmt-fe.git
cd restaurant-mgmt-fe

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
